12. Tworzenie menu poziomego i pionowego

Tematem tej lekcji jest tworzenie menu które jest nieodzowną część większości stron internetowych. Nasze menu będzie opierało się na nieuporządkowanej liście odnośników, która będzie miała przystosowany wygląd, by wyglądała jak menu. Specjalnym znacznikiem w którym zawieramy menu jest nav który jest specjalnym divem.
Zacznijmy od rzeczy której jeszcze nie wyjaśniłem w tym kursie. W arkuszu stylów możemy odnieść się do konkretnych znaczników zawartych w którymś z divów. Możemy rozróżnić dwa typu zapisu takiego odwołania. Pierwszym jest selektor potomka, w tym wypadku modyfikacja będzie dotyczyła wszystkich wybranych znaczników w danym pudełku.

Drugim typem jest selektor dziecka Taki selektor dotyczy tylko znaczników bezpośrednio zawartych w nadrzędnych znacznikach. Czyli na przykład w navie jest lista zagnieżdżona w liście, wtedy modyfikacja będzie dotyczyć tylko listy nadrzędnej.

Ważną właściwością CSS która będzie nam potrzebna do stworzenia menu jest display i jego rozszerzenia. Ta właściwość zmienia sposób wyświetlania elementu. Gdy damy mu wartość block to element będzie wyświetlony w bloku, wartość inline - element będzie wyświetlany w linii,inline-block - tworzy z niego blok liniowy. Istnieje również wartość none ale teraz nie będziemy sie nią zajmować.Spróbujmy zrobić menu pionowe:

- Przygotujmy sobie dokument HTML

nav ul{cecha1:wartość1;}
Ta modyfikacja będzie odnosiła się do wszystkich ulów w navie
nav>ul{cecha1:wartość1;}
<nav>
<ul>
<li> <a href="#"> 1 zakładka </a> </li>
 <li> <a href="#"> 2 zakładka </a> </li>
 <li> <a href="#"> 3 zakładka </a> </li>
 </ul>
</nav>

- Jeżeli chcemy to położenie menu zmieniamy przesuwając nava, natomiast my skupimy się na 3 wartościach modyfikacjach które powinniśmy zawrzeć w CSS modyfikując ul, najważniejsza to wyłączenie punktorów listy

ul{
list-style-type: none;
 margin: 0px;
 padding 0px;}

- Następnym ważnym punktem jest zrobienie z li elementu liniowego

- Możemy przejść do modyfikowania a, właśnie przy tym znaczniku zmienia się wygląd elementów menu tzn. ich kolor, obramowanie, wielkość, ważne jest by ustawić display: block

- Możemy również określić jak będzie zachowywało się nasze menu po najechaniu na element. Robimy to za pomocą funkcji hover, a w CSS odnosimy się do niej tak:

Efekt:

li{
display: inline;}
a{
text-decoration: none;
 szerokość, wysokość, wygląd
 display: block;}
a:hover{
cecha1:właściwość1;}

- Żeby stworzyć menu poziome jedyna wartość którą musisz zmienić w podpunktach powyżej to zamienienie wartości display w a na inline-block.

Efekt:

a{
display: inline-block;}
