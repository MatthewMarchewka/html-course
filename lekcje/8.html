8. Podstawowe informacje na temat arkusza stylów CSS

Po opublikowaniu 5 wersji języka HTML, wiele funkcji umożliwiających formatowanie wyglądu strony zostało przeniesione do języka CSS. Zrobienie przyjemnej dla oka strony w samym dokumencie HTML graniczy z cudem. Dlatego do modyfikowania wyglądu zawartości strony zaczęliśmy używać języka CSS3. Arkusze stylów CSS zawierają tylko reguły modyfikujące zawartość HTML
Reguła stylów w CSS składa się z: selektora czyli: znacznika, identyfikatora lub klasy. Następnie otwieramy nawias klamrowy, w którym znajduje się właściwość a po dwukropku wartość przystosowana dla tej właściwości. Przykładowa reguła stylu:

p{color:blue;}

p - selektor
color:blue; - deklaracja stylu
a całość tworzy - regułe stylu

W jednym nawiasie klamrowym możemy zawrzeć dowolną ilość deklaracji stylu dla wybranego selektora. Deklaracje w tym samym nawiasie klamrowym są oddzielone średnikami, lecz reguł nie trzeba oddzielać. Przykładowy arkusz stylów:

p{
cecha1:wartosc1;
 cecha2:wartosc2; }

 h1{
cecha1:wartosc1;
 cecha2:wartosc2; }

Istnieją trzy typy arkuszów stylu:
- zewnętrzny - w headzie dokumentu html musimy zawrzeć link do arkusza stylów (sprawdź zdjęcie poniżej). Dzięki temu możemy zastosować takie same formatowanie dla wielu różnych dokumentów html.

- wewnętrzny - Zawieramy go w headzie, pomiędzy znacznikami style. Ma taką samą konstrukcje co zewnętrzny, różnią jest, że dotyczy tylko jednego dokumentu HTML.

- lokalny - miałeś już okazję się z nim spotkać przy okazji modyfikowania list. Jego zapis wygląda jak, zapis atrybutu dla znacznika. Poniżej przykład jak powinny wyglądać takie reguły.

Przy stosowaniu arkusza stylów obowiązuje nas kaskadowość. Kaskadowość polega na tym, że im reguła stylu jest bliższa miejsca w dokumencie którego dotyczy tym jest ważniejsza dla interpretowania przeglądarki. Priorytet ważności stylów wygląda tak:

1.Lokalny arkusz stylów
2.Rozciąganie stylu (reguła dla znacznika span obejmującego większą ilość elementów)
3.Wydzielone bloki - DIV (reguła dla bloku w którym znajdują się elementy (divy przybliżę ci w następnych lekcjach))
4.Wewnętrzny arkusz stylów
5.Zewnętrzny arkusz stylów
6.Zimportowany arkusz stylów (np. z przeglądarki)
7.Domyślne ustawienia przeglądarki
Poniżej przedstawię ci przykładowy wygląd arkusza stylów wewnętrznego ponieważ to jego najłatwiej stosować na początku przy mniej zaawansowanych stronach:

<link rel="stylesheet" href="arkusz.css"/>
<style>Arkusz stylów CSS</style>
<p style="cecha1:wartosc1; cecha2:wartosc2;">Zmodyfikowany tekst</p>
<style>
p{
cecha1:wartosc1;
 cecha2:wartosc2; }

 h1{
cecha1:wartosc1;
 cecha2:wartosc2; }
 </style>
