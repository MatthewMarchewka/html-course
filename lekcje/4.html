4.Listy uporządkowane, nieuporządkowane, definicji i zagnieżdżone

Często potrzebujemy umieścić na stronie internetowej listę. Zamiast układać pod sobą akapity z tekstem, lepiej użyć do tego specjalnych znaczników które znacząco ułatwią tę czynność. Tworzenie listy polega na umieszczeniu znacznika li w znaczniku listy Do tego możemy dowolnie modyfikować punktory naszej listy za pomocą arkuszy stylów CSS które poznasz w następnym poziomie kursu.
Pierwszy znacznik to: <ul> . ten znacznik jest kontenerowy
i określa ramy listy nieuporządkowanej (jej punktory to kółka). Pomiędzy tym znacznikiem będą znajdować się elementy listy.

<ul>
Elementy listy
</ul>

Listę uporządkowaną (punktory to cyferki) tworzymy umieszczając w dokumencie znacznik <ol> .

<ol>
Elementy listy
</ol>

Elementy listy dodajemy za pomocą znacznika <li> .

<li>element listy</li>

Istnieje również jeden typ: lista definicji. Jej ramy zaznaczamy przez znacznik: <dl> .

<dl>
elementy listy definicji
</dl>

W takiej liście słowo definiowane oznaczamy znacznikiem <dt> a jego definicje w znaczniku <dd> .

<dt> Definiowane słowo </dt>
<dd> Definicja słowa </dd>

Jest również możliwość zrobienia listy zagnieżdżonej. Lista zagnieżdżona to lista w liście. Żeby stworzyć taką listę musimy wkleić listę przed zamknięciem jednego z elementu listy nadrzędnej --> </li>

<ol>
<li> 1 element </li>
<li> 2 element
<ul>
<li> 1 element 2 lista </li>
<li> 2 element 2 lista </li>
</ul>
</li>
<li> 3 element </li>
</ol>

Gdybyśmy wkleili taki kod w nasz dokument html, wyglądałby tak:

  1. 1 element
  2. 2 element
    • 1 element 2 lista
    • 2 element 2 lista
  3. 3 element

W tej części lekcji chciałbym ci pokazać jak można zmodyfikować punktory listy. Uwaga! będę wykorzystywał właściwości CSS w arkuszu stylów który będę wyjaśniał na następnym poziomie więc jeśli wydaje ci się że możesz nie zrozumieć przejdź dalej a potem wróć do tej lekcji.

Jeśli chcesz zmienić punktory listy możesz użyć właściwości list-style-type którego będziemy umieszać w atrybucie style (sprawdź przykład). Dokładny spis punktorów znajdziesz tutaj

<ul style="list-style-type: square;">
<li>1 element</li>
<li>2 element</li>
<li>3 element</li>
</ul>

Tak będzie wyglądała nasza lista:

  • 1 element
  • 2 element
  • 3 element

Możesz również ustawić jako punktor twojej listy dowolne zdjęcie, dzięki właściwości list-style-image gdzie * to nazwa zdjęcia.

<ul style="list-style-image: url(*.jpg);">
<li>1 element</li>
<li>2 element</li>
<li>3 element</li>
</ul>

Tak będzie wyglądała nasza lista:

  • 1 element
  • 2 element
  • 3 element
